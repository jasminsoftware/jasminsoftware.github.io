Logistics | Shipping Method

Overview

URL api/{tenantKey}/{orgKey}/logisticsCore/shippingMethods
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'ShippingMethod'.

GET OPERATIONS

GET ShippingMethods

Returns all Shipping Method user has access to.

Response:

If successful, this method returns a list of Shipping Method. See ShippingMethodResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /logisticsCore/shippingMethods

GET ShippingMethods (paged)

Returns a paginated list of Shipping Method user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Shipping Method. See ShippingMethodResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /logisticsCore/shippingMethods

GET ShippingMethod by identifier

Returns information for a given Shipping Method, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Shipping Method.

Response:

If successful, returns a resource of the Shipping Method. See ShippingMethodResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /logisticsCore/shippingMethods/{id}

GET ShippingMethod by key

Returns information for a given Shipping Method, by specifying its key.

The following elements are required:

shippingMethodKey (string) The key of the shipping method. The shipping method that the user can select before finalize the purchase.

Response:

If successful, returns a resource of the Shipping Method. See ShippingMethodResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /logisticsCore/shippingMethods/{shippingMethodKey}

POST OPERATIONS

POST ShippingMethods

Creates a new Shipping Method.

The following elements are required:

shippingMethodResource (ApiShippingMethodResource) In the request body, supply a resource containing the Shipping Method information.
See ApiShippingMethodResource.

Response:

If successful, returns the Shipping Method identifier in the response body and the URL in the response header Location property, or an error message. See ApiShippingMethodResource.

Response status codes:

201 The operation was successful and the Shipping Method was created.

Resource URL:

POST /logisticsCore/shippingMethods

POST Shipping Method

Inserts a new Shipping Method Detail in an existing Shipping Method.

The following elements are required:

shippingMethodKey (string) The key of the shipping method. The shipping method that the user can select before finalize the purchase.
value (ApiShippingMethodShippingMethodDetailsResource) The ApiShippingMethodShippingMethodDetailsResource instance.
See ApiShippingMethodShippingMethodDetailsResource.

Response:

If successful, this method returns an empty response body.

Response status codes:

204 The operation was successful.

Resource URL:

POST /logisticsCore/shippingMethods/{shippingMethodKey}/shippingMethodDetails

Resources


ShippingMethodResource

The following elements are required:

shippingMethodKey (string) The key of the shipping method. The shipping method that the user can select before finalize the purchase.
additionalWeight (decimal) The shipping method additional weight.
additionalWeightPrice (decimal) The shipping method additional weight price.
freeShipping (decimal) The shipping method free shipping amount.
deliveryTerm (string) The delivery term for the shipping method.
unit (string) The unit for the shipping method.
currency (string) The currency for the shipping method.

The following elements are optional:

description (string) The description for the shipping method.
deliverySpeed (int) The shipping method delivery speed in work days.
shippingMethodDetails (List of ShippingMethodDetailResource) The details of the shipping method.
See ShippingMethodDetailResource.


ShippingMethodDetailResource

The following elements are required:

from (decimal) The min weigth for the item for the shipping value.
to (decimal) The max weigth for the item for the shipping value.
shippingValue (decimal) The shipping value for the shipping method detail.
currency (string) The currency for the shipping method detail.
