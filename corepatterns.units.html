Service to manage entities of type 'Unit'.

Overview


api/{tenantKey}/{orgKey}/corePatterns/units
URL api/{tenantKey}/{orgKey}/corePatterns/units
Methods Supported GET, POST, PUT, DELETE

All Operations

This service includes the following operations:

  • GET:
    • GET Units
    • GET Units (paged)
    • GET Unit by identifier
    • GET Unit by key
    • GET Print Unit by identifier
    • GET Print Unit by key
  • POST:
    • POST Units
  • PUT:
    • PUT Set IsActive Unit
    • PUT Set UnitKey Unit
    • PUT Set Description Unit
    • PUT Set PrecisionDigits Unit
    • PUT Set ConversionFactor Unit
    • PUT Set Symbol Unit
    • PUT Set Dimension Unit
    • PUT Set IsDimensionBaseUnit Unit
    • PUT Set BaseUnit Unit
  • DELETE:
    • DELETE Unit by identifier
    • DELETE Unit by key

This service uses the following resources:

  • ApiUnitResource
  • UnitResource

GET Operations

GET Units

GET Units

Returns all Unit user has access to.

Response:

If successful, this method returns a list of Unit. See UnitResource.

Response status codes:

200 The operation was successful.

Resource URL:

api/{tenantKey}/{orgKey}/corePatterns/units
GET /corePatterns/units

GET Units (paged)

GET Units (paged)

Returns a paginated list of Unit user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results. This is a query string parameter.
pageSize (int) Specify how many items you want each page to return, up to 1000 items. This is a query string parameter.

Response:

If successful, this method returns a list of Unit. See UnitResource.

Response status codes:

200 The operation was successful.

Resource URL:

api/{tenantKey}/{orgKey}/corePatterns/units
GET /corePatterns/units

POST Operations

POST Units

POST Units

Creates a new Unit.

The following elements are required:

unitResource (ApiUnitResource) In the request body, supply a resource containing the Unit information.
See ApiUnitResource.

Response:

If successful, returns the Unit identifier in the response body and the URL in the response header Location property, or an error message. See ApiUnitResource.

Response status codes:

201 The operation was successful and the Unit was created.

Resource URL:

api/{tenantKey}/{orgKey}/corePatterns/units
POST /corePatterns/units
