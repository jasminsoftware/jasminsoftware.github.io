Materials Management | Materials Item

Overview

URL api/{tenantKey}/{orgKey}/materialsCore/materialsItems
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'MaterialsItem'.

GET OPERATIONS

GET MaterialsItems

Returns all Materials Item user has access to.

Response:

If successful, this method returns a list of Materials Item. See MaterialsItemResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /materialsCore/materialsItems/extension

GET MaterialsItems (paged)

Returns a paginated list of Materials Item user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Materials Item. See MaterialsItemResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /materialsCore/materialsItems/extension

POST OPERATIONS

POST MaterialsItems

Creates a new Materials Item.

The following elements are required:

materialsItemResource (ApiMaterialsItemResource) In the request body, supply a resource containing the Materials Item information.
See ApiMaterialsItemResource.

Response:

If successful, returns the Materials Item identifier in the response body and the URL in the response header Location property, or an error message. See ApiMaterialsItemResource.

Response status codes:

201 The operation was successful and the Materials Item was created.

Resource URL:

POST /materialsCore/materialsItems/extension

POST MaterialsItems (with ApiBaseMaterialsItem)

Creates a new Materials Item and its related Item

The following elements are required:

materialsItemResource (ApiMaterialsItemResource) The resource containing the information of the Materials Item and its related Materials Item.
See ApiMaterialsItemResource.

Response:

If successful, returns the Materials Item identifier in the response body and the URL in the response header Location property, or an error message. See ApiBaseMaterialsItemResource.

Response status codes:

201 The operation was successful and the Materials Item and its related Item were created.

Resource URL:

POST /materialsCore/materialsItems

POST Materials Item

Inserts a new Materials Item Warehouse in an existing Materials Item.

The following elements are required:

itemKey (string) The key for the item. Uniquely identifies the item within the organization.
value (ApiMaterialsItemMaterialsItemWarehousesResource) The ApiMaterialsItemMaterialsItemWarehousesResource instance.
See ApiMaterialsItemMaterialsItemWarehousesResource.

Response:

If successful, this method returns an empty response body.

Response status codes:

204 The operation was successful.

Resource URL:

POST /materialsCore/materialsItems/{itemKey}/materialsItemWarehouses
