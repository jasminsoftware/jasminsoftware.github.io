GET OPERATIONS

GET SalesSetups

Returns all Sales Setup user has access to.

GET /salesCore/salesSetups

POST OPERATIONS

POST SalesSetups

Creates a new Sales Setup.

The following elements are required:

salesSetupResource (ApiSalesSetupResource) In the request body, supply a resource containing the Sales Setup information.
See ApiSalesSetupResource.

Response:

If successful, returns the Sales Setup identifier in the response body and the URL in the response header Location property, or an error message. See ApiSalesSetupResource.

Response status codes:

201 The operation was successful and the Sales Setup was created.

Resource URL:

POST /salesCore/salesSetups
