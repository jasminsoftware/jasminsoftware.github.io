Financials | Financial Institution

Overview

URL api/{tenantKey}/{orgKey}/financialCore/financialInstitutions
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'FinancialInstitution'.

GET OPERATIONS

GET FinancialInstitutions

Returns all Financial Institution user has access to.

Response:

If successful, this method returns a list of Financial Institution. See FinancialInstitutionResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /financialCore/financialInstitutions/extension

GET FinancialInstitutions (paged)

Returns a paginated list of Financial Institution user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Financial Institution. See FinancialInstitutionResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /financialCore/financialInstitutions/extension

POST OPERATIONS

POST FinancialInstitutions

Creates a new Financial Institution.

The following elements are required:

financialInstitutionResource (ApiFinancialInstitutionResource) In the request body, supply a resource containing the Financial Institution information.
See ApiFinancialInstitutionResource.

Response:

If successful, returns the Financial Institution identifier in the response body and the URL in the response header Location property, or an error message. See ApiFinancialInstitutionResource.

Response status codes:

201 The operation was successful and the Financial Institution was created.

Resource URL:

POST /financialCore/financialInstitutions/extension

POST FinancialInstitutions (with ApiBaseFinancialInstitution)

Creates a new Financial Institution and its related Entity

The following elements are required:

financialInstitutionResource (ApiFinancialInstitutionResource) The resource containing the information of the Financial Institution and its related Financial Institution.
See ApiFinancialInstitutionResource.

Response:

If successful, returns the Financial Institution identifier in the response body and the URL in the response header Location property, or an error message. See ApiBaseFinancialInstitutionResource.

Response status codes:

201 The operation was successful and the Financial Institution and its related Entity were created.

Resource URL:

POST /financialCore/financialInstitutions
