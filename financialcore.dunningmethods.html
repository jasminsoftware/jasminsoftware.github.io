POST OPERATIONS

POST DunningMethods

Creates a new Dunning Method.

The following elements are required:

dunningMethodResource (ApiDunningMethodResource) In the request body, supply a resource containing the Dunning Method information.
See ApiDunningMethodResource.

Response:

If successful, returns the Dunning Method identifier in the response body and the URL in the response header Location property, or an error message. See ApiDunningMethodResource.

Response status codes:

201 The operation was successful and the Dunning Method was created.

Resource URL:

POST /financialCore/dunningMethods
