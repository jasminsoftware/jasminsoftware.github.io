Financials | Cash Flow Item

Overview

URL api/{tenantKey}/{orgKey}/financialCore/cashFlowItems
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'CashFlowItem'.

GET OPERATIONS

GET CashFlowItems

Returns all Cash Flow Item user has access to.

Response:

If successful, this method returns a list of Cash Flow Item. See CashFlowItemResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /financialCore/cashFlowItems

GET CashFlowItems (paged)

Returns a paginated list of Cash Flow Item user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Cash Flow Item. See CashFlowItemResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /financialCore/cashFlowItems

GET CashFlowItem by identifier

Returns information for a given Cash Flow Item, by specifying its identifier.

The following elements are required:

200 The operation was successful.

GET /financialCore/cashFlowItems/{id}

POST OPERATIONS

POST CashFlowItems

Creates a new Cash Flow Item.

The following elements are required:

cashFlowItemResource (ApiCashFlowItemResource) In the request body, supply a resource containing the Cash Flow Item information.
See ApiCashFlowItemResource.

Response:

If successful, returns the Cash Flow Item identifier in the response body and the URL in the response header Location property, or an error message. See ApiCashFlowItemResource.

Response status codes:

201 The operation was successful and the Cash Flow Item was created.

Resource URL:

POST /financialCore/cashFlowItems
