Platform | Address

Overview

URL api/{tenantKey}/{orgKey}/corePatterns/addresses
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'Address'.

GET OPERATIONS

GET Addresses

Returns all Address user has access to.

Response:

If successful, this method returns a list of Address. See AddressResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/addresses

GET Addresses (paged)

Returns a paginated list of Address user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Address. See AddressResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/addresses

GET Address by identifier

Returns information for a given Address, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Address.

Response:

If successful, returns a resource of the Address. See AddressResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/addresses/{id}

GET Address by key

Returns information for a given Address, by specifying its key.

The following elements are required:

addressKey (int) Defines the address key.

Response:

If successful, returns a resource of the Address. See AddressResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/addresses/{addressKey}

GET Print Address by identifier

Prints the specified Address.

The following elements are required:

id (string) The identifier for the Address.

The following elements are optional:

template (string) The report template.

Response:

If successful, returns a stream with the file of the printed Address.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/addresses/{id}/print

POST OPERATIONS

POST Addresses

Creates a new Address.

The following elements are required:

addressResource (ApiAddressResource) In the request body, supply a resource containing the Address information.
See ApiAddressResource.

Response:

If successful, returns the Address identifier in the response body and the URL in the response header Location property, or an error message. See ApiAddressResource.

Response status codes:

201 The operation was successful and the Address was created.

Resource URL:

POST /corePatterns/addresses

Resources


AddressResource

The following elements are required:

addressKey (int) Defines the address key.
isExternallyManaged (bool) Is Externally Managed
country (string) Defines the address country. The country provides the definition to apply to the FormattedAddress attribute.

The following elements are optional:

streetName (string) Defines the address street name.
buildingNumber (string) Defines the address building number.
postalZone (string) Defines the address postal zone.
cityName (string) Defines the address city name.
pOBox (string) Defines the address post office box.
contactName (string) Defines the address contact name.
contactTitle (string) Defines the address contact title.
formattedAddress (string) Provides the full address string formatted by the country standards. This attribute is calculated by the system.
externalId (string) External Id
externalVersion (string) External Version
region (string) Defines the address region.
postalZoneRef (string) Defines the address postal zone.
index (int) Index
