Description

GET OPERATIONS

GET Drafts

Returns all of the Drafts user has access to.

Response:

If successful, this method returns a list of Drafts. See DraftResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/drafts

GET Drafts (paged)

Returns a paginated list of Drafts user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Drafts. See DraftResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/drafts

GET Draft by identifier

Returns information for the specified Draft, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Draft.

Response:

If successful, returns a resource of the Draft. See DraftResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/drafts/{id}

GET Draft by key

Returns information for the specified Draft, by specifying its key.

The following elements are required:

draftId (Guid) Draft Id.
key (string) Key

Response:

If successful, returns a resource of the Draft. See DraftResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/drafts/{draftId}/{key}

GET Print Draft by identifier

Prints the specified Draft.

The following elements are required:

id (string) The identifier for the Draft.

The following elements are optional:

template (string) The report template.

Response:

If successful, returns a stream with the file of the printed Draft.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/drafts/{id}/print

GET Print Draft by key

Prints the specified Draft by specifying its key.

The following elements are required:

The following elements are optional:

GET /corePatterns/drafts/{draftId}/{key}/print

GET Get Total Drafts

Get the total drafts.

The following elements are required:

operation (string) Operation
state (string) State

Response:

If successful, this method returns a number.

Response status codes:

200 The operations was successful.

Resource URL:

GET /corePatterns/drafts/getTotalDrafts

GET /corePatterns/drafts/getAllDraftsTimeGroupsEx

POST OPERATIONS

POST Drafts

Creates a new Draft.

The following elements are required:

draftResource (ApiDraftResource) In the request body, supply a resource containing the Draft information.
See ApiDraftResource.

Response:

If successful, returns the Draft identifier in the response body and the URL in the response header Location property, or an error message. See ApiDraftResource.

Response status codes:

201 The operation was successful and the Draft was created.

Resource URL:

POST /corePatterns/drafts

POST Delete Drafts From Group

Custom operation 'DeleteDraftsFromGroup'.

The following elements are required:

minDate (DateTimeOffset) MinDate
maxDate (DateTimeOffset) MaxDate
operation (string) Operation
openedDraftKey (Guid) OpenedDraftKey

Response:

If successful, this method returns an empty response body.

Resource URL:

POST /corePatterns/drafts/deleteDraftsFromGroup
