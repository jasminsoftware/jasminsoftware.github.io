Service to manage entities of type 'Draft'.

Overview


api/{tenantKey}/{orgKey}/corePatterns/drafts
URL api/{tenantKey}/{orgKey}/corePatterns/drafts
Methods Supported GET, POST, PUT, DELETE

All Operations

This service includes the following operations:

  • GET:
    • GET Drafts
    • GET Drafts (paged)
    • GET Draft by identifier
    • GET Draft by key
    • GET Print Draft by identifier
    • GET Print Draft by key
    • GET Get Total Drafts
    • GET Get Drafts From Group Ex
    • GET Get All Drafts Time Groups Ex
  • POST:
    • POST Drafts
    • POST Delete Drafts From Group
  • PUT:
    • PUT Set IsActive Draft
    • PUT Set DraftId Draft
    • PUT Set Key Draft
    • PUT Set Title Draft
    • PUT Set Description Draft
    • PUT Set Resource Draft
    • PUT Set ControllerName Draft
    • PUT Set ParentId Draft
    • PUT Set OperationType Draft
    • PUT Set State Draft
    • PUT Set ProcessColor Draft
    • PUT Set ProcessCaption Draft
    • PUT Set ModuleName Draft
    • PUT Set ServiceName Draft
    • PUT Set ResourceObject Draft
    • PUT Set Operation Draft
    • PUT Set OperationName Draft
  • DELETE:
    • DELETE Draft by identifier
    • DELETE Draft by key

GET Drafts

GET Drafts (paged)

POST Operations

POST Drafts

POST Drafts

Creates a new Draft.

The following elements are required:

draftResource (ApiDraftResource) In the request body, supply a resource containing the Draft information.
See ApiDraftResource.

Response:

If successful, returns the Draft identifier in the response body and the URL in the response header Location property, or an error message. See ApiDraftResource.

Response status codes:

201 The operation was successful and the Draft was created.

Resource URL:

api/{tenantKey}/{orgKey}/corePatterns/drafts
POST /corePatterns/drafts

POST Delete Drafts From Group

operation (string) The operation. This is a query string parameter.
