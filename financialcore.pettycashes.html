Resources


PettyCashResource

The following elements are required:

pettyCashKey (string) The petty cash's key.
company (string) The petty cash's company.
allowForeignCurrency (bool) Indicated whether the patty cash allows currencies other than the base currency.
allowNegativeBalances (bool) Indicates whether the petty cash allows negative balances.

The following elements are optional:

description (string) The petty cash’s description.
responsible (string) The person held resposible for the petty cash.
validFrom (DateTime) The start date for validity period.
validTo (DateTime) The end date for validity period.
pettyCashBalances (List of PettyCashBalanceResource) The petty cash's balances list.
See PettyCashBalanceResource.


PettyCashBalanceResource

The following elements are required:

currentBalance (decimal) The current balance in money.
currency (string) The currency in which the petty cash balance is represented.
paymentMethod (string) The associated payment method.
