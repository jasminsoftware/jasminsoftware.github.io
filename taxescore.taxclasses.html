Taxes | Tax Class

Service to manage entities of type 'TaxClass'.

Overview


api/{tenantKey}/{orgKey}/taxesCore/taxClasses
URL api/{tenantKey}/{orgKey}/taxesCore/taxClasses
Methods Supported GET, POST, PUT, DELETE

All Operations

This service includes the following operations:

  • GET:
    • GET TaxClasses
    • GET TaxClasses (paged)
    • GET TaxClass by identifier
    • GET TaxClass by key
    • GET Print TaxClass by identifier
    • GET Print TaxClass by key
    • GET Original Print TaxClassOriginal by identifier
    • GET Original Print TaxClass by key
  • POST:
    • POST TaxClasses
  • PUT:
    • PUT Set IsActive TaxClass
    • PUT Set TaxClassKey TaxClass
    • PUT Set Description TaxClass
    • PUT Set ExternalId TaxClass
    • PUT Set ExternalVersion TaxClass
    • PUT Set IsExternallyManaged TaxClass
  • DELETE:
    • DELETE TaxClass by identifier
    • DELETE TaxClass by key

  • TaxClassResource
