Overview

URL api/{tenantKey}/{orgKey}/salesCore/quotationTypes
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'QuotationType'.

GET OPERATIONS

GET QuotationTypes

Returns all Quotation Type user has access to.

Response:

If successful, this method returns a list of Quotation Type. See QuotationTypeResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/quotationTypes

GET QuotationTypes (paged)

Returns a paginated list of Quotation Type user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Quotation Type. See QuotationTypeResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/quotationTypes

GET QuotationType by identifier

Returns information for a given Quotation Type, by specifying its identifier.

The following elements are required:

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/quotationTypes/{id}

POST OPERATIONS

POST QuotationTypes

Creates a new Quotation Type.

The following elements are required:

quotationTypeResource (ApiQuotationTypeResource) In the request body, supply a resource containing the Quotation Type information.
See ApiQuotationTypeResource.

Response:

If successful, returns the Quotation Type identifier in the response body and the URL in the response header Location property, or an error message. See ApiQuotationTypeResource.

Response status codes:

201 The operation was successful and the Quotation Type was created.

Resource URL:

POST /salesCore/quotationTypes

POST Quotation Type

Inserts a new Quotation Type Serie in an existing Quotation Type.

The following elements are required:

typeKey (string) The quotation type key.
companyKey (string) Company key to which the requested Quotation Type is associated to.
value (ApiQuotationTypeDocumentTypeSeriesResource) The ApiQuotationTypeDocumentTypeSeriesResource instance.
See ApiQuotationTypeDocumentTypeSeriesResource.

Response:

If successful, this method returns an empty response body.

Response status codes:

204 The operation was successful.

Resource URL:

POST /salesCore/quotationTypes/{companyKey}/{typeKey}/documentTypeSeries

Resources


QuotationTypeResource

The following elements are required:

company (string) The order type company. The company code is the central organizational unit within the system.
typeKey (string) The quotation type key.

The following elements are optional:

description (string) The quotation type description.
documentTypeSeries (List of QuotationTypeSerieResource) The quotation type details.
See QuotationTypeSerieResource.
defaultOrderType (string) Indicates the default order type when the quotations are processed.
fiscalDocumentType (string) The fiscal document type.
contactType (string) The contact type for the order type.
notification (string) The default notification for documents with this order type.


QuotationTypeSerieResource

The following elements are required:

isDefault (bool) Indicates if this is the default serie for the document.
taxIncluded (bool) When checked, the document unit prices will include taxes.
numberOfCopies (int) Number of copies of the document to print.
serie (string) Range of identification values that group a set of product entities or objects.

The following elements are optional:

currentNumber (int) Current document number.
currentDate (DateTime) The current date for the order type serie.


ApiQuotationTypeResource

The following elements are required:

company (string) The order type company. The company code is the central organizational unit within the system.
typeKey (string) The quotation type key.

The following elements are optional:

description (string) The quotation type description.
documentTypeSeries (List of ApiQuotationTypeDocumentTypeSeriesResource) The quotation type details.
See ApiQuotationTypeDocumentTypeSeriesResource.
defaultOrderType (string) Indicates the default order type when the quotations are processed.
fiscalDocumentType (string) The fiscal document type.
contactType (string) The contact type for the order type.
notification (string) The default notification for documents with this order type.


ApiQuotationTypeDocumentTypeSeriesResource

The following elements are required:

isDefault (bool) Indicates if this is the default serie for the document.
taxIncluded (bool) When checked, the document unit prices will include taxes.
numberOfCopies (int) Number of copies of the document to print.
serie (string) Range of identification values that group a set of product entities or objects.
