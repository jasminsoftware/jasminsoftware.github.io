Master Data Business Entities | Item

Overview

URL api/{tenantKey}/{orgKey}/businessCore/items
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'Item'. The item is also designated as product or good, an item is the element that uniquely identifies the product or service involved in a transaction.

GET OPERATIONS

GET Items

Returns all Item user has access to.

Response:

If successful, this method returns a list of Item. See ItemResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /businessCore/items

GET Items (paged)

Returns a paginated list of Item user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Item. See ItemResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /businessCore/items

GET Item by identifier

Returns information for a given Item, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Item.

Response:

If successful, returns a resource of the Item. See ItemResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /businessCore/items/{id}

GET Item by key

Returns information for a given Item, by specifying its key.

The following elements are required:

itemKey (string) The key for the item. Uniquely identifies the item within the organization.

Response:

If successful, returns a resource of the Item. See ItemResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /businessCore/items/{itemKey}

GET Print Item by identifier

Prints the specified Item.

The following elements are required:

id (string) The identifier for the Item.

The following elements are optional:

template (string) The report template.

Response:

If successful, returns a stream with the file of the printed Item.

Response status codes:

200 The operation was successful.

Resource URL:

GET /businessCore/items/{id}/print

GET Item Image.

Returns the Image of the specified Item. This method returns an image stream.

The following elements are required:

id (string) The identifier for the Item.
fileName (string) File name.

Response:

If successful, returns a stream with the image file of the Item Image.

Response status codes:

200 The operation was successful.

Resource URL:

GET /businessCore/items/{id}/image

POST OPERATIONS

POST Items

Creates a new Item.

The following elements are required:

itemResource (ApiItemResource) In the request body, supply a resource containing the Item information.
See ApiItemResource.

Response:

If successful, returns the Item identifier in the response body and the URL in the response header Location property, or an error message. See ApiItemResource.

Response status codes:

201 The operation was successful and the Item was created.

Resource URL:

POST /businessCore/items
