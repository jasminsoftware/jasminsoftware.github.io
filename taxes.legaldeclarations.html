Taxes | SAF-T (PT)

Overview

URL api/{tenantKey}/{orgKey}/taxes/legalDeclarations
Methods Supported GET, DELETE, POST
Description Service to manage entities of type 'LegalDeclaration'.

GET OPERATIONS

GET LegalDeclarations

Returns all Legal Statement user has access to.

Response:

If successful, this method returns a list of Legal Statement. See LegalDeclarationResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxes/legalDeclarations

GET LegalDeclarations (paged)

Returns a paginated list of Legal Statement user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Legal Statement. See LegalDeclarationResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxes/legalDeclarations

GET LegalDeclaration by identifier

Returns information for a given Legal Statement, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Legal Statement.

Response:

If successful, returns a resource of the Legal Statement. See LegalDeclarationResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxes/legalDeclarations/{id}

GET LegalDeclaration by key

Returns information for a given Legal Statement, by specifying its key.

The following elements are required:

companyKey (string) Company key to which the requested Legal Statement is associated to.
legalDeclarationKey (string) The legal declaration key. By default, it is a sequential number.

Response:

If successful, returns a resource of the Legal Statement. See LegalDeclarationResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxes/legalDeclarations/{companyKey}/{legalDeclarationKey}

GET Print LegalDeclaration by identifier

Prints the specified LegalDeclaration.

The following elements are required:

id (string) The identifier for the LegalDeclaration.

The following elements are optional:

template (string) The report template.

Response:

If successful, returns a stream with the file of the printed LegalDeclaration.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxes/legalDeclarations/{id}/print

GET Print LegalDeclaration by key

Prints the specified LegalDeclaration by specifying its key.

The following elements are required:

companyKey (string) Company key to which the requested LegalDeclaration is associated to.
legalDeclarationKey (string) The legal declaration key. By default, it is a sequential number.

The following elements are optional:

template (string) The report template.

Response:

If successful, returns a stream with the file of the printed LegalDeclaration.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxes/legalDeclarations/{companyKey}/{legalDeclarationKey}/print

POST OPERATIONS

POST Legal Declaration Saft

Custom operation 'LegalDeclarationSaft'.

The following elements are required:

company (string) Company
legalDeclarationPurpose (int) 1 - InvoiceDeclaration, 3- MovementOfGoods, 4 - Auditing
startDate (DateTime?) StartDate
endDate (DateTime?) EndDate

Response:

If successful, this method returns an unique identifier (GUID).

Resource URL:

POST /taxes/legalDeclarations/legalDeclarationSaft/{company}/{legalDeclarationPurpose}

DELETE OPERATIONS

DELETE LegalDeclaration by identifier

Deletes the specified Legal Statement by its identifier.

The following elements are required:

id (string) The identifier for the Legal Statement.

Response:

If successful, returns an empty response body.

Response status codes:

204 The Legal Statement was successfully deleted.
400 The Legal Statement does not exist.

Resource URL:

DELETE /taxes/legalDeclarations/{id}

DELETE LegalDeclaration by key

Deletes the specified Legal Statement by its key.

The following elements are required:

companyKey (string) Company key to which the requested Legal Statement is associated to.
legalDeclarationKey (string) The legal declaration key. By default, it is a sequential number.

Response:

If successful, returns an empty response body.

Response status codes:

204 The Legal Statement was successfully deleted.
400 The Legal Statement does not exist.

Resource URL:

DELETE /taxes/legalDeclarations/{companyKey}/{legalDeclarationKey}

Resources


LegalDeclarationResource

The following elements are required:

legalDeclarationKey (string) The legal declaration key. By default, it is a sequential number.
documentDate (DateTime) Date
startDate (DateTime) The legal declaration initial date. Sets the date from which documents should be included on SAF-T.
endDate (DateTime) The legal declaration final date. Sets the date up to which documents should be included on SAF-T.
totalCredit (decimal) The legal declaration total credit. Indicates the total amount of documents of type credit included on SAF-T.
totalDebit (decimal) The legal declaration total debit. Indicates the total amount of documents of type debit included on SAF-T.
company (string) The legal declaration company. The company code is the central organizational unit within the system.
status (enum LegalDeclarationState) The legal declaration status.
See LegalDeclarationState.
currency (string) The legal declaration currency. Indicates the money unit used by the generated document.
auditFileVersion (enum AuditFileVersion) The audit file version. Indicates the SAF-T version used in the legal declaration.
See AuditFileVersion.
legalDeclarationPurpose (enum LegalDeclarationPurpose) The legal declaration purpose.
See LegalDeclarationPurpose.

The following elements are optional:

legalDeclarationLines (List of LegalDeclarationLineResource) The legal declaration lines.
See LegalDeclarationLineResource.
note (string) The legal declaration reference.
remarks (string) The legal declaration remarks.
isPrinted (bool) Indicates if the legal declaration has been printed. Internally determined.
numberOfDocuments (int) The number of documents included on SAF-T.
messages (List of LegalDeclarationMessageResource) The legal declaration messages.
See LegalDeclarationMessageResource.
saftDocument (string) Saft Document


LegalDeclarationLineResource

The following elements are required:

sourceDoc (string) Source document using the format {..}. Indicates the original document included on SAF-T.
sourceDocId (Guid) Source document Id. Indicates the original document included on SAF-T.
sourceSchemaEntityId (Guid) Source schema entity Id. Indicates the schema entity of the original document included on SAF-T.
taxableAmount (decimal) The taxable amount of the original document included on SAF-T.
taxAmount (decimal) The tax amount of the original document included on SAF-T.
currency (string) Indicates the currency of the original document.

The following elements are optional:

documentDate (DateTime) The date of the original document included on SAF-T.
companyTaxId (string) Unique code assigned by a country's tax authority to establish an entity's fiscal identification and responsibilities. Identify the entity of the original document.
party (string) Indicates the entity of the original document.


LegalDeclarationMessageResource

The following elements are optional:

message (string) The legal declaration message.
details (string) The legal declaration message details.
