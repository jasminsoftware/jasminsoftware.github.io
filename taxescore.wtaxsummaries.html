GET OPERATIONS

GET WTaxSummaries

Returns all Withholding Tax Summary user has access to.

Response:

If successful, this method returns a list of Withholding Tax Summary. See WTaxSummaryResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxesCore/wTaxSummaries

Returns a paginated list of Withholding Tax Summary user has access to.

The following elements are optional:

GET /taxesCore/wTaxSummaries

POST OPERATIONS

POST WTaxSummaries

Creates a new Withholding Tax Summary.

The following elements are required:

wTaxSummaryResource (ApiWTaxSummaryResource) In the request body, supply a resource containing the Withholding Tax Summary information.
See ApiWTaxSummaryResource.

Response:

If successful, returns the Withholding Tax Summary identifier in the response body and the URL in the response header Location property, or an error message. See ApiWTaxSummaryResource.

Response status codes:

201 The operation was successful and the Withholding Tax Summary was created.

Resource URL:

POST /taxesCore/wTaxSummaries
