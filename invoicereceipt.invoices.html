GET Invoice by key

GET Invoice by key

Returns information for a given Invoice, by specifying its key.

The following elements are required:

companyKey (string) Company key to which the requested Invoice is associated to.
documentType (string) Defines the invoice's purpose. Distinguishes the business transactions to be posted. Part of the document natural key. The default invoice document type is defined in the purchases setup configuration setting.
serie (string) Defines the sequence to generate the series number. Part of the document natural key. The default serie is defined on the invoice document type.
seriesNumber (int) Sequential number internally determined that uniquely identifies the document within the invoice document type and serie. Part of the document natural key.

Response:

If successful, returns a resource of the Invoice. See InvoiceResource.

Response status codes:

200 The operation was successful.
