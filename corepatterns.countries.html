Overview

URL api/{tenantKey}/{orgKey}/corePatterns/countries
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'Country'.

GET OPERATIONS

GET Countries

Returns all Country user has access to.

Response:

If successful, this method returns a list of Country. See CountryResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/countries

GET Countries (paged)

Returns a paginated list of Country user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Country. See CountryResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/countries

GET Country by identifier

Returns information for a given Country, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Country.

Response:

If successful, returns a resource of the Country. See CountryResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/countries/{id}

GET Country by key

Returns information for a given Country, by specifying its key.

The following elements are required:

Response:

If successful, returns a resource of the Country. See CountryResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/countries/{countryKey}

GET Print Country by identifier

The following elements are required:

POST OPERATIONS

POST Countries

Creates a new Country.

The following elements are required:

countryResource (ApiCountryResource) In the request body, supply a resource containing the Country information.
See ApiCountryResource.

Response:

If successful, returns the Country identifier in the response body and the URL in the response header Location property, or an error message. See ApiCountryResource.

Response status codes:

201 The operation was successful and the Country was created.

Resource URL:

POST /corePatterns/countries
