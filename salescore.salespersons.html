Sales | Salesperson

Overview

URL api/{tenantKey}/{orgKey}/salesCore/salespersons
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'Salesperson'.

GET OPERATIONS

GET Salespersons

Returns all Salesperson user has access to.

Response:

If successful, this method returns a list of Salesperson. See SalespersonResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/salespersons/extension

GET Salespersons (paged)

Returns a paginated list of Salesperson user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Salesperson. See SalespersonResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/salespersons/extension

POST OPERATIONS

POST Salespeople

Creates a new Salesperson.

The following elements are required:

salespersonResource (ApiSalespersonResource) In the request body, supply a resource containing the Salesperson information.
See ApiSalespersonResource.

Response:

If successful, returns the Salesperson identifier in the response body and the URL in the response header Location property, or an error message. See ApiSalespersonResource.

Response status codes:

201 The operation was successful and the Salesperson was created.

Resource URL:

POST /salesCore/salespersons/extension

POST Salespeople (with ApiBaseSalesperson)

Creates a new Salesperson and its related Entity

The following elements are required:

salespersonResource (ApiSalespersonResource) The resource containing the information of the Salesperson and its related Salesperson.
See ApiSalespersonResource.

Response:

If successful, returns the Salesperson identifier in the response body and the URL in the response header Location property, or an error message. See ApiBaseSalespersonResource.

Response status codes:

201 The operation was successful and the Salesperson and its related Entity were created.

Resource URL:

POST /salesCore/salespersons
