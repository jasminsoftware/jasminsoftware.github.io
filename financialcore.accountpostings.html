Financials | Account Posting

Overview

URL api/{tenantKey}/{orgKey}/financialCore/accountPostings
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'AccountPosting'.

GET OPERATIONS

GET AccountPostings

Returns all Account Posting user has access to.

Response:

If successful, this method returns a list of Account Posting. See AccountPostingResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /financialCore/accountPostings

GET AccountPostings (paged)

Returns a paginated list of Account Posting user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Account Posting. See AccountPostingResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /financialCore/accountPostings

GET AccountPosting by identifier

Returns information for a given Account Posting, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Account Posting.

Response:

If successful, returns a resource of the Account Posting. See AccountPostingResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /financialCore/accountPostings/{id}

GET AccountPosting by key

Returns information for a given Account Posting, by specifying its key.

The following elements are required:

POST OPERATIONS

POST AccountPostings

Creates a new Account Posting.

The following elements are required:

accountPostingResource (ApiAccountPostingResource) In the request body, supply a resource containing the Account Posting information.
See ApiAccountPostingResource.

Response:

If successful, returns the Account Posting identifier in the response body and the URL in the response header Location property, or an error message. See ApiAccountPostingResource.

Response status codes:

201 The operation was successful and the Account Posting was created.

Resource URL:

POST /financialCore/accountPostings
