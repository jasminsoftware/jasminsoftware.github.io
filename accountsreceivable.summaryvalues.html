Overview


api/{tenantKey}/{orgKey}/accountsReceivable/summaryValues
URL api/{tenantKey}/{orgKey}/accountsReceivable/summaryValues
Methods Supported GET

All Operations

This service includes the following operations:

  • GET:
    • GET Get Sales Open List Data
    • GET Get Sales Open Card Data
    • GET Get Sales Open List Count

All Resources

This service uses the following resources:

  • NamedSummarizedValue
  • SalesOpenListData

GET Operations

GET Get Sales Open List Data

GET Get Sales Open List Data

Custom operation 'GetSalesOpenListData'.

The following elements are required:

company (string) Company This is a query string parameter.
code (string) Code This is a query string parameter.

Response:

When successful, the response body contains a list of resources of type 'SalesOpenListData'. See SalesOpenListData.

Response status codes:

200 OK. The operation was completed successfully.
400 BadRequest. The request parameters are invalid.

Resource URL:

api/{tenantKey}/{orgKey}/accountsReceivable/summaryValues/getSalesOpenListData/{company}/{code}?company={company}&code={code}
GET /accountsReceivable/summaryValues/getSalesOpenListData/{company}/{code}?company={company}&code={code}
