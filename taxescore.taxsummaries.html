Taxes | Open Tax Item

Overview

URL api/{tenantKey}/{orgKey}/taxesCore/taxSummaries
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'TaxSummary'.

GET OPERATIONS

GET TaxSummaries

Returns all Open Tax Item user has access to.

Response:

If successful, this method returns a list of Open Tax Item. See TaxSummaryResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxesCore/taxSummaries

GET TaxSummaries (paged)

Returns a paginated list of Open Tax Item user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Open Tax Item. See TaxSummaryResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxesCore/taxSummaries

GET TaxSummary by identifier

Returns information for a given Open Tax Item, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Open Tax Item.

Response:

If successful, returns a resource of the Open Tax Item. See TaxSummaryResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxesCore/taxSummaries/{id}

GET TaxSummary by key

Returns information for a given Open Tax Item, by specifying its key.

The following elements are required:

taxPostingKey (int) Chave única desta tabela

Response:

If successful, returns a resource of the Open Tax Item. See TaxSummaryResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxesCore/taxSummaries/{taxPostingKey}

GET Print TaxSummary by identifier

Prints the specified TaxSummary.

The following elements are required:

id (string) The identifier for the TaxSummary.

The following elements are optional:

template (string) The report template.

Response:

If successful, returns a stream with the file of the printed TaxSummary.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxesCore/taxSummaries/{id}/print

GET Print TaxSummary by key

Prints the specified TaxSummary by specifying its key.

The following elements are required:

taxPostingKey (int) Chave única desta tabela

The following elements are optional:

Response:

Resource URL:

GET /taxesCore/taxSummaries/{taxPostingKey}/print

POST OPERATIONS

POST TaxSummaries

Creates a new Open Tax Item.

The following elements are required:

taxSummaryResource (ApiTaxSummaryResource) In the request body, supply a resource containing the Open Tax Item information.
See ApiTaxSummaryResource.

Response:

If successful, returns the Open Tax Item identifier in the response body and the URL in the response header Location property, or an error message. See ApiTaxSummaryResource.

Response status codes:

201 The operation was successful and the Open Tax Item was created.

Resource URL:

POST /taxesCore/taxSummaries
