Goods Receipt | Process Order

Overview

URL api/{tenantKey}/{orgKey}/goodsReceipt/processOrders
Methods Supported GET, POST
Description

GET OPERATIONS

GET Process Order

Returns a paginated list of available entities that can be processed by the GoodsReceiptNotes service.

The following elements are required:

pageIndex (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.
company (string) The company code is the central organizational unit within the system.

The following elements are optional:

warehouse (string) The order warehouse. The default is defined on the materials item.
deliveryDateFrom (DateTime?) The initial date to deliver.
deliveryDateUntil (DateTime?) The final date to deliver.
loadingPoint (string) The address where the good is loaded.
unloadingPoint (string) The address where the good is unloaded.
party (string) The customer from whom the goods are ordered.
goodsReceiptNoteType (string) The goods receipt type.

Response:

If successful, returns a list of resources available to process. See GoodsReceiptOrderLine.

Response status codes:

200 The operation was successful.

Resource URL:

GET /goodsReceipt/processOrders/{pageIndex}/{pageSize}

POST OPERATIONS

POST Process Order

Process the selected resources by the GoodsReceiptNotes service.

The following elements are required:

list (GoodsReceiptOrderLine) List of GoodsReceiptOrderLine to process.
See GoodsReceiptOrderLine.
companyKey (string) Company key to which the requested GoodsReceiptOrderLine is associated to.

Response:

If successful, returns an empty body response and the URL of the processed items in the response location header.

Response status codes:

201 The operation was successful.

Resource URL:

POST /goodsReceipt/processOrders/{companyKey}

Resources


GoodsReceiptOrder

The following elements are required:

selectedCount (int) Number of lines selected. Internally calculated.
company (string) The company code is the central organizational unit within the system.

The following elements are optional:

warehouse (string) The order warehouse. The default is defined on the materials item.
deliveryDateFrom (DateTime) The initial date to deliver.
deliveryDateUntil (DateTime) The final date to deliver.
loadingPoint (string) The address where the good is loaded.
unloadingPoint (string) The address where the good is unloaded.
party (string) The customer from whom the goods are ordered.
goodsReceiptNoteType (string) The goods receipt type.


GoodsReceiptOrderLine

The following elements are required:

selected (bool) Indicates if the line has been selected.

The following elements are optional:

description (string) The goods receipt oders line item description. The default is defined on the order item.
originalQuantity (decimal) The initial goods quantity.
openQuantity (decimal) The quantity that was not yet processed.
quantity (decimal) The quantity available to deliver.
unit (string) The goods receipt order line item order unit. The default order unit is defined on the order item.
sourceDocKey (string) The source document id, when the order was transformed. Internally determined.
sourceDocLineNumber (int) The source document line number, when the order was transformed. Internally determined.
receiptNotificationId (Guid) The receipt notification id for the goods receipt order line.
receiptNotificationLineId (Guid) The receipt notification line id for the goods receipt order line.
party (string) The customer from whom the goods are ordered.
item (string) The original order item.
deliveryDate (DateTime) The delivery date.
