Logistics | Delivery Type

Overview

URL api/{tenantKey}/{orgKey}/logisticsCore/deliveryTypes
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'DeliveryType'.

GET OPERATIONS

GET DeliveryTypes

Returns all Delivery Type user has access to.

Response:

If successful, this method returns a list of Delivery Type. See DeliveryTypeResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /logisticsCore/deliveryTypes

GET DeliveryTypes (paged)

Returns a paginated list of Delivery Type user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Delivery Type. See DeliveryTypeResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /logisticsCore/deliveryTypes

GET DeliveryType by identifier

Returns information for a given Delivery Type, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Delivery Type.

Response:

If successful, returns a resource of the Delivery Type. See DeliveryTypeResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /logisticsCore/deliveryTypes/{id}

GET DeliveryType by key

Returns information for a given Delivery Type, by specifying its key.

The following elements are required:

companyKey (string) Company key to which the requested Delivery Type is associated to.
typeKey (string) The key for the delivery type.

Response:

If successful, returns a resource of the Delivery Type. See DeliveryTypeResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /logisticsCore/deliveryTypes/{companyKey}/{typeKey}

POST OPERATIONS

POST DeliveryTypes

Creates a new Delivery Type.

The following elements are required:

deliveryTypeResource (ApiDeliveryTypeResource) In the request body, supply a resource containing the Delivery Type information.
See ApiDeliveryTypeResource.

Response:

If successful, returns the Delivery Type identifier in the response body and the URL in the response header Location property, or an error message. See ApiDeliveryTypeResource.

Response status codes:

201 The operation was successful and the Delivery Type was created.

Resource URL:

POST /logisticsCore/deliveryTypes

POST Delivery Type

Inserts a new Delivery Type Serie in an existing Delivery Type.

The following elements are required:

typeKey (string) The key for the delivery type.
companyKey (string) Company key to which the requested Delivery Type is associated to.
value (ApiDeliveryTypeDocumentTypeSeriesResource) The ApiDeliveryTypeDocumentTypeSeriesResource instance.
See ApiDeliveryTypeDocumentTypeSeriesResource.

Response:

If successful, this method returns an empty response body.

Response status codes:

204 The operation was successful.

Resource URL:

POST /logisticsCore/deliveryTypes/{companyKey}/{typeKey}/documentTypeSeries

Resources


DeliveryTypeResource

The following elements are required:

typeKey (string) The key for the delivery type.
company (string) The delivery type company. The company for the company type. The company code is the central organizational unit within the system.
isPortugueseCompany (bool) Internal flag indicating if the delivery type company country is Portugal.
isInternal (bool) Is Internal

The following elements are optional:

description (string) The description for the delivery type.
fiscalDocumentType (string) The fiscal document type for the delivery type.
documentTypeSeries (List of DeliveryTypeSerieResource) List of available series for this document type.
See DeliveryTypeSerieResource.
contactType (string) The contact type for the delivery type.
notification (string) The notification for the delivery type.


DeliveryTypeSerieResource

The following elements are required:

numberOfCopies (int) Number of copies of the document to print.
isDefault (bool) Indicates if this is the default serie for the document.
isManual (bool) Indicates if the delivery type is serie is manual.
isExternal (bool) Indicates if the delivery type is serie is external.
serie (string) The serie for the delivery type serie.
communicationType (enum CommunicationType) The communication type to comunicate the document to the tax authority.
See CommunicationType.

The following elements are optional:

currentDate (DateTime) The current date for the order delivery serie.
currentNumber (int) The current delivery type serie number.


ApiDeliveryTypeResource

The following elements are required:

typeKey (string) The key for the delivery type.
company (string) The delivery type company. The company for the company type. The company code is the central organizational unit within the system.
isPortugueseCompany (bool) Internal flag indicating if the delivery type company country is Portugal.
isInternal (bool) Is Internal

The following elements are optional:

description (string) The description for the delivery type.
fiscalDocumentType (string) The fiscal document type for the delivery type.
documentTypeSeries (List of ApiDeliveryTypeDocumentTypeSeriesResource) List of available series for this document type.
See ApiDeliveryTypeDocumentTypeSeriesResource.
contactType (string) The contact type for the delivery type.
notification (string) The notification for the delivery type.


ApiDeliveryTypeDocumentTypeSeriesResource

The following elements are required:

numberOfCopies (int) Number of copies of the document to print.
isDefault (bool) Indicates if this is the default serie for the document.
isManual (bool) Indicates if the delivery type is serie is manual.
isExternal (bool) Indicates if the delivery type is serie is external.
