URL api/{tenantKey}/{orgKey}/billing/processOrders
Methods Supported GET, POST
Description

GET OPERATIONS

GET Process Order

Returns a paginated list of available entities that can be processed by the Invoices service.

The following elements are required:

pageIndex (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

The following elements are optional:

company (string) Company
accountingCustomerParty (string) Accounting Customer Party
invoiceType (string) Invoice Type
billingRequestDateFrom (DateTime?) Billing Request Date From
billingRequestDateUntil (DateTime?) Billing Request Date Until
currency (string) Currency
sourceDoc (string) Source Doc
deliveryKey (string) Delivery Key

Response:

If successful, returns a list of resources available to process. See BillingOrderLine.

Response status codes:

200 The operation was successful.

Resource URL:

GET /billing/processOrders/{pageIndex}/{pageSize}

POST OPERATIONS

POST Process Order

Process the selected resources by the Invoices service.

The following elements are required:

list (BillingOrderLine) List of BillingOrderLine to process.
See BillingOrderLine.
companyKey (string) Company key to which the requested BillingOrderLine is associated to.

Response:

If successful, returns an empty body response and the URL of the processed items in the response location header.

Response status codes:

201 The operation was successful.

Resource URL:

POST /billing/processOrders/{companyKey}
