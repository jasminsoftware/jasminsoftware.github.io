URL api/{tenantKey}/{orgKey}/financialCore/financialParties
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'FinancialParty'.

POST OPERATIONS

POST FinancialParties

Creates a new Financial Party.

The following elements are required:

financialPartyResource (ApiFinancialPartyResource) In the request body, supply a resource containing the Financial Party information.
See ApiFinancialPartyResource.

Response:

If successful, returns the Financial Party identifier in the response body and the URL in the response header Location property, or an error message. See ApiFinancialPartyResource.

Response status codes:

201 The operation was successful and the Financial Party was created.

Resource URL:

POST /financialCore/financialParties/extension

POST FinancialParties (with ApiBaseFinancialParty)

Creates a new Financial Party and its related Entity

The following elements are required:

financialPartyResource (ApiFinancialPartyResource) The resource containing the information of the Financial Party and its related Financial Party.
See ApiFinancialPartyResource.

Response:

If successful, returns the Financial Party identifier in the response body and the URL in the response header Location property, or an error message. See ApiBaseFinancialPartyResource.

201 The operation was successful and the Financial Party and its related Entity were created.

Resource URL:

POST /financialCore/financialParties
