Overview


api/{tenantKey}/{orgKey}/accountsPayable/summaryValues
URL api/{tenantKey}/{orgKey}/accountsPayable/summaryValues
Methods Supported GET

All Operations

This service includes the following operations:

  • GET:
    • GET Get Purchases Open List Data
    • GET Get Purchases Open Card Data
    • GET Get Purchases Open List Count

All Resources

This service uses the following resources:

  • NamedSummarizedValue
  • PurchasesOpenListData

GET Operations

GET Get Purchases Open List Data

GET Get Purchases Open List Data

Custom operation 'GetPurchasesOpenListData'.

The following elements are required:

company (string) Company This is a query string parameter.
code (string) Code This is a query string parameter.

Response:

When successful, the response body contains a list of resources of type 'PurchasesOpenListData'. See PurchasesOpenListData.

Response status codes:

200 OK. The operation was completed successfully.
400 BadRequest. The request parameters are invalid.

Resource URL:

api/{tenantKey}/{orgKey}/accountsPayable/summaryValues/getPurchasesOpenListData/{company}/{code}?company={company}&code={code}
GET /accountsPayable/summaryValues/getPurchasesOpenListData/{company}/{code}?company={company}&code={code}
