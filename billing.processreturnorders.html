GET OPERATIONS

GET Process Return Order

GET Process Return Order

Returns a paginated list of available entities that can be processed by the Memos service.

The following elements are required:

pageIndex (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

The following elements are optional:

company (string) Company This is a query string parameter.
accountingCustomerParty (string) Accounting Customer Party This is a query string parameter.
memoType (string) Memo Type This is a query string parameter.
billingRequestDateFrom (DateTime?) Billing Request Date From This is a query string parameter.
billingRequestDateUntil (DateTime?) Billing Request Date Until This is a query string parameter.
currency (string) Currency This is a query string parameter.
sourceDoc (string) Source Doc This is a query string parameter.
goodsReceiptNoteKey (string) Goods Receipt Note Key This is a query string parameter.

Response:

If successful, returns a list of resources available to process. See BillingReturnOrderLine.

Response status codes:

200 The operation was successful.

Resource URL:

api/{tenantKey}/{orgKey}/billing/processReturnOrders/{pageIndex}/{pageSize}
GET /billing/processReturnOrders/{pageIndex}/{pageSize}

POST OPERATIONS

POST Process Return Order

POST Process Return Order

Process the selected resources by the Memos service.

The following elements are required:

list (BillingReturnOrderLine) List of BillingReturnOrderLine to process.
See BillingReturnOrderLine.
companyKey (string) Company key to which the requested BillingReturnOrderLine is associated to.

Response:

If successful, returns an empty body response and the URL of the processed items in the response location header.

Response status codes:

201 The operation was successful.

Resource URL:

api/{tenantKey}/{orgKey}/billing/processReturnOrders/{companyKey}
POST /billing/processReturnOrders/{companyKey}
