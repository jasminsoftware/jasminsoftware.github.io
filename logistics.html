Logistics

The Logistics API exposes the logistics setup entities used by processes from other areas, such as sales or purchases.

Services

This area of the Web API includes the following services:

  • Delivery Term
  • Delivery Type
  • Goods Receipt Type
  • Stock Settings
  • Memo Reason
  • Shipping Method
