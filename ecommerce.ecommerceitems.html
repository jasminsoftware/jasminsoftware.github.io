POST OPERATIONS

POST ECommerceItems

POST ECommerceItems

Creates a new ECommerce Item.

The following elements are required:

eCommerceItemResource (ApiECommerceItemResource) In the request body, supply a resource containing the ECommerce Item information.
See ApiECommerceItemResource.

Response:

If successful, returns the ECommerce Item identifier in the response body and the URL in the response header Location property, or an error message. See ApiECommerceItemResource.

Response status codes:

201 The operation was successful and the ECommerce Item was created.

Resource URL:

api/{tenantKey}/{orgKey}/eCommerce/eCommerceItems/extension
POST /eCommerce/eCommerceItems/extension

Resources


ECommerceItemResource

The following elements are required:

description (string) This is the public description displayed in the store.
weight (decimal) The item weight used to calculate the shipping cost.
unit (string) The unit used on the shipping method. Only mass units are available.
itemKey (string) The key for the item. Uniquely identifies the item within the organization.

The following elements are optional:

availability (bool) If this atribute is set to true then the item is displayed in the store.
technicalFeatures (string) Use this attribute to set all the detail item technical features.
image1 (string) Item alternative Image.
image2 (string) Item alternative Image.
image3 (string) Item alternative Image.
image4 (string) Item alternative Image.
itemCategory (string) The category where the item will be displaid. If an item has an associated category only itens with category will be displayed.
