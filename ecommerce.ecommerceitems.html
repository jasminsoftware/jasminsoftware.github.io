Online Store | ECommerce Item

Service to manage entities of type 'ECommerceItem'.

This service extends Items.

Resources


ApiBaseECommerceItemResource

ApiBaseECommerceItemResource

The following elements are required:

description (string) This is the public description displayed in the store.
weight (decimal) The item's weight used to calculate the shipping cost.
unit (string) The unit used on the shipping method. Only mass units are available.
itemKey (string) The key for the item. Uniquely identifies the item within the organization.
isExternallyManaged (bool) A flag indicating if the item is externally managed.
baseUnit (string) The base unit for the item.
itemType (enum ItemType) The item type for the item.
See ItemType.

The following elements are optional:

availability (bool) If this atribute is set to true then the item is displayed in the store.
technicalFeatures (string) Use this attribute to set all the detail item technical features.
image1 (string) The item's alternative Image.
image2 (string) The item's alternative Image.
image3 (string) The item's alternative Image.
image4 (string) The item's alternative Image.
itemCategory (string) The category where the item will be displaid. If an item has an associated category only itens with category will be displayed.
complementaryDescription (string) The extended description for the item.
assortment (string) The assortment for the item.
barcode (string) The barcode for the item.
brand (string) The brand for the item.
image (string) The picture for the item.
remarks (string) The remarks for the item.
externalId (string) The external identifier for the item.
externalVersion (string) The external version for the item.
brandModel (string) The brand model for the item.
