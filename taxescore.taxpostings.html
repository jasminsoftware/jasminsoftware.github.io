Overview

URL api/{tenantKey}/{orgKey}/taxesCore/taxPostings
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'TaxPosting'.

GET OPERATIONS

GET TaxPostings

Returns all Tax Posting user has access to.

Response:

If successful, this method returns a list of Tax Posting. See TaxPostingResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxesCore/taxPostings

GET TaxPostings (paged)

Returns a paginated list of Tax Posting user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Tax Posting. See TaxPostingResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxesCore/taxPostings

GET TaxPosting by identifier

Returns information for a given Tax Posting, by specifying its identifier.

The following elements are required:

Resource URL:

POST OPERATIONS

POST TaxPostings

Creates a new Tax Posting.

The following elements are required:

taxPostingResource (ApiTaxPostingResource) In the request body, supply a resource containing the Tax Posting information.
See ApiTaxPostingResource.

Response:

If successful, returns the Tax Posting identifier in the response body and the URL in the response header Location property, or an error message. See ApiTaxPostingResource.

Response status codes:

201 The operation was successful and the Tax Posting was created.

Resource URL:

POST /taxesCore/taxPostings
