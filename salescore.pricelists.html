Sales | Price List

Overview

URL api/{tenantKey}/{orgKey}/salesCore/priceLists
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'PriceList'.

GET OPERATIONS

GET PriceLists

Returns all Price List user has access to.

Response:

If successful, this method returns a list of Price List. See PriceListResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/priceLists

GET PriceLists (paged)

Returns a paginated list of Price List user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Price List. See PriceListResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/priceLists

GET PriceList by identifier

Returns information for a given Price List, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Price List.

Response:

If successful, returns a resource of the Price List. See PriceListResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/priceLists/{id}

GET PriceList by key

Returns information for a given Price List, by specifying its key.

The following elements are required:

priceListKey (string) Price list key

Response:

If successful, returns a resource of the Price List. See PriceListResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/priceLists/{priceListKey}

GET Print PriceList by identifier

Prints the specified PriceList.

The following elements are required:

id (string) The identifier for the PriceList.

The following elements are optional:

template (string) The report template.

Response:

If successful, returns a stream with the file of the printed PriceList.

Response status codes:

200 The operation was successful.

GET /salesCore/priceLists/{id}/print

POST OPERATIONS

POST PriceLists

Creates a new Price List.

The following elements are required:

priceListResource (ApiPriceListResource) In the request body, supply a resource containing the Price List information.
See ApiPriceListResource.

Response:

If successful, returns the Price List identifier in the response body and the URL in the response header Location property, or an error message. See ApiPriceListResource.

Response status codes:

201 The operation was successful and the Price List was created.

Resource URL:

POST /salesCore/priceLists
