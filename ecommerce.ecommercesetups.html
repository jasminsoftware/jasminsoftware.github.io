Resources


ECommerceSetupResource

The following elements are required:

storeSetupKey (string) The store identifier.
company (string) The company where the orders will be save.
orderType (string) The default order type used when the customer complete the order process.
priceList (string) The price list used on the store. Only items with this price list will be listed.
paymentMethod (string) The default payment method use on the checkout process.
partyTaxSchema (string) The party tax schema used by the customer.
shippingMethod (string) The default shipping method use on the checkout process.
salesChannel (string) This field allows you to organize your sales according to their origin.
paymentTerm (string) The default payment term use on the checkout process.
customerGroup (string) The customer group contains the default information that allows create a new customer when the user complete the registration process.
salesItem (string) The item that will be added to the document with the shipping costs.

The following elements are optional:

description (string) The store description.
aboutus (string) Aboutus information that is displayed on the store footer page.
shippingPayment (string) The Shipping and Payment accepted by the store. This information is displayed on the store footer page.
termsAndConditions (string) The Terms and Conditions defined by the store. This information is displayed on the store footer page.
privacyPolicy (string) The Privacy and Policy defined by the store. This information is displayed on the store footer page.
exchangesAndReturns (string) The Exchanges and Returns policy defined by the store. This information is displayed on the store footer page.
pageInMaintenance (bool) When this flag is set to true the page under maintenance message will be show to the customer. The page content is still available to the administrator.
