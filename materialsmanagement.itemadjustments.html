POST ItemAdjustments

POST ItemAdjustments

Creates a new Item Adjustment.

See ApiItemAdjustmentResource.

If successful, returns the Item Adjustment identifier in the response body and the URL in the response header Location property, or an error message. See ApiItemAdjustmentResource.

Response status codes:

api/{tenantKey}/{orgKey}/materialsManagement/itemAdjustments
POST /materialsManagement/itemAdjustments
