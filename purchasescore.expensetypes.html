POST OPERATIONS

POST ExpenseTypes

Creates a new ExpenseType.

The following elements are required:

expenseTypeResource (ApiExpenseTypeResource) In the request body, supply a resource containing the ExpenseType information.
See ApiExpenseTypeResource.

Response:

If successful, returns the ExpenseType identifier in the response body and the URL in the response header Location property, or an error message. See ApiExpenseTypeResource.

Response status codes:

201 The operation was successful and the ExpenseType was created.

Resource URL:

POST /purchasesCore/expenseTypes

POST ExpenseType

Inserts a new Expense Type Serie in an existing ExpenseType.

The following elements are required:

typeKey (string) The key for the expense type.
companyKey (string) Company key to which the requested ExpenseType is associated to.
value (ApiExpenseTypeDocumentTypeSeriesResource) The ApiExpenseTypeDocumentTypeSeriesResource instance.
See ApiExpenseTypeDocumentTypeSeriesResource.

Response:

If successful, this method returns an empty response body.

Response status codes:

204 The operation was successful.

Resource URL:

POST /purchasesCore/expenseTypes/{companyKey}/{typeKey}/documentTypeSeries

Resources


ExpenseTypeResource

The following elements are required:

company (string) Company
typeKey (string) The key for the expense type.

The following elements are optional:

description (string) The description for the expense type.
defaultCashFlowItem (string) Default Cash Flow Item
documentTypeSeries (Array of ExpenseTypeSerieResource) Document Type Series
See ExpenseTypeSerieResource.


ExpenseTypeSerieResource

The following elements are required:

serie (string) Serie
isDefault (bool) Indicates if this is the default serie for the document.
numberOfCopies (int) Number of copies of the document to print.

The following elements are optional:

taxIncluded (bool) When checked, the document unit prices will include taxes.
currentNumber (int) Current document number.
currentDate (DateTime) The current date for the invoice type serie.
