Purchases | Memo Type

Overview

URL api/{tenantKey}/{orgKey}/purchasesCore/memoTypes
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'MemoType'.

GET OPERATIONS

GET MemoTypes

Returns all Memo Type user has access to.

Response:

If successful, this method returns a list of Memo Type. See MemoTypeResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /purchasesCore/memoTypes

GET MemoTypes (paged)

Returns a paginated list of Memo Type user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Memo Type. See MemoTypeResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /purchasesCore/memoTypes

GET MemoType by identifier

Returns information for a given Memo Type, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Memo Type.

Response:

If successful, returns a resource of the Memo Type. See MemoTypeResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /purchasesCore/memoTypes/{id}

GET MemoType by key

Returns information for a given Memo Type, by specifying its key.

The following elements are required:

companyKey (string) Company key to which the requested Memo Type is associated to.
typeKey (string) The key for the memo type.

Response:

If successful, returns a resource of the Memo Type. See MemoTypeResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /purchasesCore/memoTypes/{companyKey}/{typeKey}

GET Print MemoType by identifier

POST OPERATIONS

POST MemoTypes

Creates a new Memo Type.

The following elements are required:

memoTypeResource (ApiMemoTypeResource) In the request body, supply a resource containing the Memo Type information.
See ApiMemoTypeResource.

Response:

If successful, returns the Memo Type identifier in the response body and the URL in the response header Location property, or an error message. See ApiMemoTypeResource.

Response status codes:

201 The operation was successful and the Memo Type was created.

Resource URL:

POST /purchasesCore/memoTypes

POST Memo Type

Inserts a new Memo Type Serie in an existing Memo Type.

The following elements are required:

typeKey (string) The key for the memo type.
companyKey (string) Company key to which the requested Memo Type is associated to.
value (ApiMemoTypeDocumentTypeSeriesResource) The ApiMemoTypeDocumentTypeSeriesResource instance.
See ApiMemoTypeDocumentTypeSeriesResource.

Response:

If successful, this method returns an empty response body.

Response status codes:

204 The operation was successful.

Resource URL:

POST /purchasesCore/memoTypes/{companyKey}/{typeKey}/documentTypeSeries

Resources


MemoTypeResource

The following elements are required:

company (string) The company code is the central organizational unit within the system.
typeKey (string) The key for the memo type.
accountingNature (enum AccountingNature) The nature for the memo type. It can be "credit" or "debit".
See AccountingNature.

The following elements are optional:

description (string) The description for the memo type.
documentTypeSeries (List of MemoTypeSerieResource) List of available series for this document type.
See MemoTypeSerieResource.
accountingSchema (enum AccountingSchema) The entity's accounting organization method. This organization can follow the rules of "Cash" or "Accrual".
See AccountingSchema.
contactType (string) The contact type for the memo type.
notification (string) The default notification for documents with this memo type.


MemoTypeSerieResource

The following elements are required:

isDefault (bool) Indicates if this is the default serie for the document.
numberOfCopies (int) Number of copies of the document to print.
serie (string) Range of identification values that group a set of product entities or objects.

The following elements are optional:

currentNumber (int) Current document number.
currentDate (DateTime) The current date for the order type serie.
taxIncluded (bool) When checked, the document unit prices will include taxes.


ApiMemoTypeResource

The following elements are required:

company (string) The company code is the central organizational unit within the system.
typeKey (string) The key for the memo type.
accountingNature (enum AccountingNature) The nature for the memo type. It can be "credit" or "debit".
See AccountingNature.

The following elements are optional:

description (string) The description for the memo type.
documentTypeSeries (List of ApiMemoTypeDocumentTypeSeriesResource) List of available series for this document type.
See ApiMemoTypeDocumentTypeSeriesResource.
accountingSchema (enum AccountingSchema) The entity's accounting organization method. This organization can follow the rules of "Cash" or "Accrual".
See AccountingSchema.
contactType (string) The contact type for the memo type.
notification (string) The default notification for documents with this memo type.


ApiMemoTypeDocumentTypeSeriesResource

The following elements are required:

isDefault (bool) Indicates if this is the default serie for the document.
numberOfCopies (int) Number of copies of the document to print.

The following elements are optional:
