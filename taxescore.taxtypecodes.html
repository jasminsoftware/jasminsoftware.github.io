GET OPERATIONS

GET TaxTypeCodes

Returns all Tax Type Code user has access to.

Response:

If successful, this method returns a list of Tax Type Code. See TaxTypeCodeResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxesCore/taxTypeCodes

GET TaxTypeCodes (paged)

Returns a paginated list of Tax Type Code user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.

Response:

If successful, this method returns a list of Tax Type Code. See TaxTypeCodeResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxesCore/taxTypeCodes

POST OPERATIONS

POST TaxTypeCodes

Creates a new Tax Type Code.

The following elements are required:

taxTypeCodeResource (ApiTaxTypeCodeResource) In the request body, supply a resource containing the Tax Type Code information.
See ApiTaxTypeCodeResource.

Response:

If successful, returns the Tax Type Code identifier in the response body and the URL in the response header Location property, or an error message. See ApiTaxTypeCodeResource.

Response status codes:

201 The operation was successful and the Tax Type Code was created.

Resource URL:

POST /taxesCore/taxTypeCodes

POST Tax Type Code

Inserts a new Tax Code Rate in an existing Tax Type Code.

The following elements are required:

taxCodeKey (string) The tax code key.
value (ApiTaxTypeCodeTaxCodeRatesResource) The ApiTaxTypeCodeTaxCodeRatesResource instance.
See ApiTaxTypeCodeTaxCodeRatesResource.

Response:

If successful, this method returns an empty response body.

Response status codes:

204 The operation was successful.

Resource URL:

POST /taxesCore/taxTypeCodes/{taxCodeKey}/taxCodeRates
