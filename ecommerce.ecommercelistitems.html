Online Store | ECommerce List Items

Get all eCommerce items that are available at the store.

Overview


api/{tenantKey}/{orgKey}/eCommerce/eCommerceListItems
URL api/{tenantKey}/{orgKey}/eCommerce/eCommerceListItems
Methods Supported GET

All Operations

This service includes the following operations:

  • GET:
    • GET Get ECommerce Items
    • GET Get E Commerce Item

All Resources

This service uses the following resources:

  • ECommerceItemDetail
  • ECommerceListItem

GET Operations

GET Get ECommerce Items

GET Get ECommerce Items

Retuns all eCommerce items.

The following elements are required:

companyKey (string) CompanyKey This is a query string parameter.

Response:

When successful, the response body contains a list of resources of type 'ECommerceListItem'. See ECommerceListItem.

Response status codes:

200 OK. The operation was completed successfully.
400 BadRequest. The request parameters are invalid.

Resource URL:

api/{tenantKey}/{orgKey}/eCommerce/eCommerceListItems/getECommerceItems?companyKey={companyKey}
GET /eCommerce/eCommerceListItems/getECommerceItems?companyKey={companyKey}

GET Get E Commerce Item

GET Get E Commerce Item

Get the Ecommerce item detail.

The following elements are required:

companyKey (string) CompanyKey This is a query string parameter.
itemKey (string) ItemKey This is a query string parameter.

Response:

When successful, the response body contains a resource of type 'ECommerceItemDetail'. See ECommerceItemDetail.

Response status codes:

200 OK. The operation was completed successfully.
400 BadRequest. The request parameters are invalid.

Resource URL:

api/{tenantKey}/{orgKey}/eCommerce/eCommerceListItems/getECommerceItem?companyKey={companyKey}&itemKey={itemKey}
GET /eCommerce/eCommerceListItems/getECommerceItem?companyKey={companyKey}&itemKey={itemKey}
