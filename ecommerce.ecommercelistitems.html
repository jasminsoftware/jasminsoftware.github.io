Online Store | ECommerce List Items

Overview

URL api/{tenantKey}/{orgKey}/eCommerce/eCommerceListItems
Methods Supported GET
Description

GET OPERATIONS

GET Get ECommerce Items

Retuns all eCommerce items.

The following elements are required:

companyKey (string) CompanyKey

Response:

If successful, this method returns a list of ECommerceListItem resources. See ECommerceListItem.

Response status codes:

200 The operations was successful.

Resource URL:

GET /eCommerce/eCommerceListItems/getECommerceItems

GET Get E Commerce Item

Get the Ecommerce item detail.

The following elements are required:

companyKey (string) CompanyKey
itemKey (string) ItemKey

Response:

If successful, this method returns a ECommerceItemDetail. See ECommerceItemDetail.

Response status codes:

200 The operations was successful.

Resource URL:

GET /eCommerce/eCommerceListItems/getECommerceItem
