Shipping | Shipping Request

Overview

URL api/{tenantKey}/{orgKey}/shipping/shippingRequests
Methods Supported GET
Description Service to manage entities of type 'ShippingRequest'.

GET OPERATIONS

GET ShippingRequests

Returns all Shipping Request user has access to.

Response:

If successful, this method returns a list of Shipping Request. See ShippingRequestResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /shipping/shippingRequests

GET ShippingRequests (paged)

Returns a paginated list of Shipping Request user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Shipping Request. See ShippingRequestResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /shipping/shippingRequests

GET ShippingRequest by identifier

Returns information for a given Shipping Request, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Shipping Request.

Response:

If successful, returns a resource of the Shipping Request. See ShippingRequestResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /shipping/shippingRequests/{id}

GET ShippingRequest by key

Returns information for a given Shipping Request, by specifying its key.

The following elements are required:

companyKey (string) Company key to which the requested Shipping Request is associated to.
shippingRequestKey (int) Sequential number internally determined that uniquely identifies the shipping request. The document natural key.

Response:

If successful, returns a resource of the Shipping Request. See ShippingRequestResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /shipping/shippingRequests/{companyKey}/{shippingRequestKey}

GET Print ShippingRequest by identifier

Prints the specified ShippingRequest.

The following elements are required:

The following elements are optional:

Response:

If successful, returns a stream with the file of the printed ShippingRequest.

GET /shipping/shippingRequests/{id}/print
