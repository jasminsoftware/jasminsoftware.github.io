Purchases | Memo

Overview

URL api/{tenantKey}/{orgKey}/invoiceReceipt/memos
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'Memo'.

GET OPERATIONS

GET Memos

Returns all Memo user has access to.

Response:

If successful, this method returns a list of Memo. See MemoResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /invoiceReceipt/memos

GET Memos (paged)

Returns a paginated list of Memo user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Memo. See MemoResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /invoiceReceipt/memos

GET Memo by identifier

Returns information for a given Memo, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Memo.

Response:

If successful, returns a resource of the Memo. See MemoResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /invoiceReceipt/memos/{id}

GET Memo by key

Returns information for a given Memo, by specifying its key.

The following elements are required:

companyKey (string) Company key to which the requested Memo is associated to.
documentType (string) Defines the memo's purpose. Distinguishes the business transactions to be posted. Part of the document natural key. The default memo document type is defined in the purchases setup configuration setting.
serie (string) Defines the sequence to generate the series number. Part of the document natural key. The default serie is defined on the memo document type.
seriesNumber (int) Sequential number internally determined that uniquely identifies the document within the memo document type and serie. Part of the document natural key.

Response:

If successful, returns a resource of the Memo. See MemoResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /invoiceReceipt/memos/{companyKey}/{documentType}/{serie}/{seriesNumber}

GET Print Memo by identifier

Prints the specified Memo.

The following elements are required:

id (string) The identifier for the Memo.

The following elements are optional:

template (string) The report template.

Response:

If successful, returns a stream with the file of the printed Memo.

Response status codes:

200 The operation was successful.

Resource URL:

GET /invoiceReceipt/memos/{id}/print

GET Print Memo by key

Prints the specified Memo by specifying its key.

The following elements are required:

companyKey (string) Company key to which the requested Memo is associated to.

POST Memos

The following elements are required:

memoResource (ApiMemoResource) In the request body, supply a resource containing the Memo information.
See ApiMemoResource.

Response:

If successful, returns the Memo identifier in the response body and the URL in the response header Location property, or an error message. See ApiMemoResource.

201 The operation was successful and the Memo was created.

POST /invoiceReceipt/memos

The default is defined on the supplier party.
currency (string) Money unit used by the system, such as Euros (EUR) or United States Dollars (USD). This value is important for currency exchanges or to set the number of decimal places used to represent values. The default is defined on the supplier party.

In other words, the rate at which the document's currency can be exchanged for the company's base currency. The default is inferred using the defined exchange rates on the system for the document date and currency.

wTaxTotal (decimal) Total withholding tax value of goods and services. Internally calculated.
totalLiability (decimal) Document total liability. Internally calculated.
documentStatus (enum DocumentStatus) The document status. Internally determined.

fromPicking (bool) Indicates if the memo was result of a picking process. Internally determined.

nature (enum Nature) The original document nature (Debit or Credit). Internally determined.

notification (string) The memo's notification configuration. Inferred from the invoice type.
emailTo (string) The email(s) to send the memo notification. Inferred from the notification.
