GET OPERATIONS

GET FiscalClassifications

Returns all Tax Classification user has access to.

Response:

If successful, this method returns a list of Tax Classification. See FiscalClassificationResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxesCore/fiscalClassifications

GET FiscalClassifications (paged)

Returns a paginated list of Tax Classification user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Tax Classification. See FiscalClassificationResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /taxesCore/fiscalClassifications

GET FiscalClassification by identifier

POST OPERATIONS

POST FiscalClassifications

Creates a new Tax Classification.

The following elements are required:

fiscalClassificationResource (ApiFiscalClassificationResource) In the request body, supply a resource containing the Tax Classification information.
See ApiFiscalClassificationResource.

Response:

If successful, returns the Tax Classification identifier in the response body and the URL in the response header Location property, or an error message. See ApiFiscalClassificationResource.

Response status codes:

201 The operation was successful and the Tax Classification was created.

Resource URL:

POST /taxesCore/fiscalClassifications
