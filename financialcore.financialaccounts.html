Financials | Financial Account

Overview

URL api/{tenantKey}/{orgKey}/financialCore/financialAccounts
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'FinancialAccount'.

GET OPERATIONS

GET FinancialAccounts

Returns all Bank Account user has access to.

Response:

If successful, this method returns a list of Bank Account. See FinancialAccountResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /financialCore/financialAccounts

GET FinancialAccounts (paged)

Returns a paginated list of Bank Account user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Bank Account. See FinancialAccountResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /financialCore/financialAccounts

GET FinancialAccount by identifier

Returns information for a given Bank Account, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Bank Account.

Response:

If successful, returns a resource of the Bank Account. See FinancialAccountResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /financialCore/financialAccounts/{id}

GET FinancialAccount by key

Returns information for a given Bank Account, by specifying its key.

The following elements are required:

financialAccountKey (string) The financial account key.

Response:

If successful, returns a resource of the Bank Account. See FinancialAccountResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /financialCore/financialAccounts/{financialAccountKey}

GET Print FinancialAccount by identifier

Prints the specified FinancialAccount.

The following elements are required:

id (string) The identifier for the FinancialAccount.

The following elements are optional:

template (string) The report template.

Response:

If successful, returns a stream with the file of the printed FinancialAccount.

Response status codes:

200 The operation was successful.

Resource URL:

GET /financialCore/financialAccounts/{id}/print

GET Print FinancialAccount by key

Prints the specified FinancialAccount by specifying its key.

The following elements are required:

The following elements are optional:

Response:

Response status codes:

Resource URL:

GET /financialCore/financialAccounts/{financialAccountKey}/print

POST OPERATIONS

POST FinancialAccounts

Creates a new Bank Account.

The following elements are required:

financialAccountResource (ApiFinancialAccountResource) In the request body, supply a resource containing the Bank Account information.
See ApiFinancialAccountResource.

Response:

If successful, returns the Bank Account identifier in the response body and the URL in the response header Location property, or an error message. See ApiFinancialAccountResource.

Response status codes:

201 The operation was successful and the Bank Account was created.

Resource URL:

POST /financialCore/financialAccounts

Resources


FinancialAccountResource

The following elements are required:

company (string) The financial account's associated company.
accountNumber (string) The account number.
financialAccountKey (string) The financial account key.
reconciliationTolerance (decimal) The reconciliation tolerance, by default 2.
currency (string) The currency which the accounts operates.
country (string) The country where the account's institution is based.

The following elements are optional:

description (string) The financial account description.
iBAN (string) The finacial account's International Bank Account Number.
sWIFT (string) The society for Worldwide Interbank Financial Telecommunication number.
branch (string) The branch name.
validFrom (DateTime) The account's lower date validation.
validTo (DateTime) The account's upper date validation.
partyTaxSchema (string) The party tax schema.
financialInstitution (string) The associated financial institution.
partyWithholdingTaxSchema (string) The party withholding tax schema.
