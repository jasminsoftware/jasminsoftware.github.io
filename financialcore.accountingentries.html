GET Get Accounting Summaries

GET Get Accounting Summaries

Gets the accounting entries summary of each month.

The following elements are required:

startDate (DateTime?) StartDate This is a query string parameter.
endDate (DateTime?) EndDate This is a query string parameter.

Response:

When successful, the response body contains a list of resources of type 'AccountingSummary'. See AccountingSummary.

Response status codes:

200 OK. The operation was completed successfully.
400 BadRequest. The request parameters are invalid.

Resource URL:

api/{tenantKey}/{orgKey}/financialCore/accountingEntries/getAccountingSummaries?startDate={startDate}&endDate={endDate}
GET /financialCore/accountingEntries/getAccountingSummaries?startDate={startDate}&endDate={endDate}
