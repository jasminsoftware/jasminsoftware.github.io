Materials Management

The Materials Management API exposes the setup and management of base entities such as materials items or warehouses and inventory management transactions, which allows you to perform item adjustments like stock corrections for specific warehouses.

Services

This area of the Web API includes the following services:

  • Adjustment Reason
  • Item Adjustment
  • Item Subtype
  • Materials Item
  • Stock Settings
  • Stock Transfer Order
  • Warehouse
