Taxes | Obligation

Overview

URL api/{tenantKey}/{orgKey}/taxes/obligations
Methods Supported POST
Description Service to manage entities of type 'Obligation'.

POST OPERATIONS

POST Register Fiscal Obligation

Custom operation 'RegisterFiscalObligation'.

The following elements are required:

obligationResource (ObligationPublicResource) ObligationResource

Response:

If successful, this method returns an empty response body.

Resource URL:

POST /taxes/obligations/registerFiscalObligation

Resources


ObligationPublicResource

The following elements are required:

companyTaxId (string) The company tax id. Unique code assigned by a country's tax authority to establish an entity's fiscal identification and responsibilities.
dueDate (DateTime) The obligation due date. Date in which the payment/receivable is due. In some countries it is also known as law date.
period (string) The obligation period. The period to which the fiscal obligation refers.
taxCode (string) The tax code. Code that identifies the type of tax applied.
taxDescription (string) The tax description.
year (int) The obligation year. The year to which the tax obligation relates.
payableAmount (decimal) Payable Amount
receivableAmount (decimal) Receivable Amount
