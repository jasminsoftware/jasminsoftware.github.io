Overview

URL api/{tenantKey}/{orgKey}/corePatterns/companies
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'Company'.

GET OPERATIONS

GET Companies

Returns all Company user has access to.

Response:

If successful, this method returns a list of Company. See CompanyResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/companies

GET Companies (paged)

Returns a paginated list of Company user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Company. See CompanyResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/companies

GET Company by identifier

Returns information for a given Company, by specifying its identifier.

The following elements are required:

Response:

If successful, returns a resource of the Company. See CompanyResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/companies/{id}

POST OPERATIONS

POST Companies

Creates a new Company.

The following elements are required:

companyResource (ApiCompanyResource) In the request body, supply a resource containing the Company information.
See ApiCompanyResource.

Response:

If successful, returns the Company identifier in the response body and the URL in the response header Location property, or an error message. See ApiCompanyResource.

Response status codes:

201 The operation was successful and the Company was created.

Resource URL:

POST /corePatterns/companies

POST Is Multiple Company

Custom operation 'IsMultipleCompany'.

Response:

If successful, this method returns a boolean.

Resource URL:

POST /corePatterns/companies/isMultipleCompany

CompanyResource

The following elements are required:

companyKey (string) The company key.
name (string) The company name.
companyTaxID (string) The company tax identifier.
country (string) The company country.
baseCurrency (string) The company base currency.

The following elements are optional:

isExternal (bool) Is External
showExchangeRateReversed (bool) Indicates wheter the exchange rate for the base currency is presented with reversed logic.
telephone (string) The company telephone number formatted in the local format, including the area code, country code, and extension.
teleFax (string) The company fax number formatted in the local format, including the area code, country code, and extension.
mobile (string) The mobile phone number formatted in the local format, including the area code, country code, and extension.
electronicMail (string) The e-mail address of the comapny.
equityCapital (string) Equity capital is defined as the amount of capital provided by the company's owner(s).
registrationOffice (string) Registration Office
registrationNumber (string) Registration Number
buildingNumber (string) The company building number.
streetName (string) The company street name.
cityName (string) The company city name.
postalZone (string) The company postal zone.
address (string) The company address.
logo (string) The company logo.
