GET OPERATIONS

GET Deliveries

GET Deliveries

Returns all Delivery user has access to.

Response:

If successful, this method returns a list of Delivery. See DeliveryResource.

Response status codes:

200 The operation was successful.

Resource URL:

api/{tenantKey}/{orgKey}/shipping/deliveries
GET /shipping/deliveries

GET Deliveries (paged)

GET Deliveries (paged)

Returns a paginated list of Delivery user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results. This is a query string parameter.
pageSize (int) Specify how many items you want each page to return, up to 1000 items. This is a query string parameter.

Response:

If successful, this method returns a list of Delivery. See DeliveryResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /shipping/deliveries

GET Delivery by identifier

Returns information for a given Delivery, by specifying its identifier.

The following elements are required:

Response:

If successful, returns a resource of the Delivery. See DeliveryResource.

Response status codes:

200 The operation was successful.

Resource URL:

api/{tenantKey}/{orgKey}/shipping/deliveries/{id}
GET /shipping/deliveries/{id}

GET Delivery by key

GET Delivery by key

Returns information for a given Delivery, by specifying its key.

The following elements are required:

companyKey (string) Company key to which the requested Delivery is associated to.
seriesNumber (int) Sequential number internally determined that uniquely identifies the document within the delivery type and serie. Part of the document natural key.
serie (string) Defines the sequence to generate the series number. Part of the document natural key. The default serie is defined on the delivery type.
documentType (string) Defines the delivery's purpose. Distinguishes the business transactions to be posted. Part of the document natural key. The default delivery type is defined in the sales setup configuration setting.

Response:

If successful, returns a resource of the Delivery. See DeliveryResource.

Response status codes:

200 The operation was successful.

Resource URL:

api/{tenantKey}/{orgKey}/shipping/deliveries/{companyKey}/{seriesNumber}/{serie}/{documentType}
GET /shipping/deliveries/{companyKey}/{seriesNumber}/{serie}/{documentType}
