Sales | Sales Channel

Overview

URL api/{tenantKey}/{orgKey}/salesCore/salesChannels
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'SalesChannel'.

GET OPERATIONS

GET SalesChannels

Returns all Sales Channel user has access to.

Response:

If successful, this method returns a list of Sales Channel. See SalesChannelResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/salesChannels

GET SalesChannels (paged)

Returns a paginated list of Sales Channel user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Sales Channel. See SalesChannelResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/salesChannels

GET SalesChannel by identifier

Returns information for a given Sales Channel, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Sales Channel.

Response:

If successful, returns a resource of the Sales Channel. See SalesChannelResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/salesChannels/{id}

GET SalesChannel by key

Returns information for a given Sales Channel, by specifying its key.

The following elements are required:

Response:

Response status codes:

200 The operation was successful.

Resource URL:

GET /salesCore/salesChannels/{salesChannelKey}

GET Print SalesChannel by identifier

Prints the specified SalesChannel.

The following elements are required:

The following elements are optional:

POST OPERATIONS

POST SalesChannels

Creates a new Sales Channel.

The following elements are required:

salesChannelResource (ApiSalesChannelResource) In the request body, supply a resource containing the Sales Channel information.
See ApiSalesChannelResource.

Response:

If successful, returns the Sales Channel identifier in the response body and the URL in the response header Location property, or an error message. See ApiSalesChannelResource.

Response status codes:

201 The operation was successful and the Sales Channel was created.

Resource URL:

POST /salesCore/salesChannels
