All Operations

This service includes the following operations:

  • GET:
    • GET Get Product News
  • POST:
    • POST Mark News As Viewed

All Resources

This service uses the following resources:

  • ProductNews

GET Operations

GET Get Product News

GET Get Product News

Custom operation 'GetProductNews'.

The following elements are required:

version (string) Version This is a query string parameter.

Response:

When successful, the response body contains a resource of type 'ProductNews'. See ProductNews.

Response status codes:

200 OK. The operation was completed successfully.
400 BadRequest. The request parameters are invalid.

Resource URL:

api/{tenantKey}/{orgKey}/sales/productNews/getProductNews?version={version}
GET /sales/productNews/getProductNews?version={version}

POST Operations

POST Mark News As Viewed

POST Mark News As Viewed

Custom operation 'MarkNewsAsViewed'.

The following elements are required:

version (string) Version This is a query string parameter.

Response:

When successful, the response body is empty.

Resource URL:

api/{tenantKey}/{orgKey}/sales/productNews/markNewsAsViewed?version={version}
POST /sales/productNews/markNewsAsViewed?version={version}
