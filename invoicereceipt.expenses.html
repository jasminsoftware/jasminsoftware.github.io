GET Expense by key

GET Expense by key

Returns information for a given Expense, by specifying its key.

The following elements are required:

companyKey (string) Company key to which the requested Expense is associated to.
documentType (string) Document Type
serie (string) Defines the sequence to generate the series number. Part of the document natural key. The default serie is defined on the expense type.
seriesNumber (int) Sequential number internally determined that uniquely identifies the document within the expense type and serie. Part of the document natural key.

Response:

If successful, returns a resource of the Expense. See ExpenseResource.

Response status codes:

200 The operation was successful.

POST Expenses

POST Expenses

Creates a new Expense.

The following elements are required:

See ApiExpenseResource.

Response:

If successful, returns the Expense identifier in the response body and the URL in the response header Location property, or an error message. See ApiExpenseResource.

Response status codes:

201 The operation was successful and the Expense was created.
