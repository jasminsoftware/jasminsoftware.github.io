Platform | Currency

Overview

URL api/{tenantKey}/{orgKey}/corePatterns/currencies
Methods Supported GET, POST, PUT, DELETE
Description Service to manage entities of type 'Currency'.

GET OPERATIONS

GET Currencies

Returns all Currency user has access to.

Response:

If successful, this method returns a list of Currency. See CurrencyResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/currencies

GET Currencies (paged)

Returns a paginated list of Currency user has access to.

The following elements are optional:

page (int) The number of the page you want to consume the results.
pageSize (int) Specify how many items you want each page to return, up to 1000 items.

Response:

If successful, this method returns a list of Currency. See CurrencyResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/currencies

GET Currency by identifier

Returns information for a given Currency, by specifying its identifier.

The following elements are required:

id (string) The identifier for the Currency.

Response:

If successful, returns a resource of the Currency. See CurrencyResource.

Response status codes:

200 The operation was successful.

Resource URL:

GET /corePatterns/currencies/{id}

POST OPERATIONS

POST Currencies

Creates a new Currency.

The following elements are required:

currencyResource (ApiCurrencyResource) In the request body, supply a resource containing the Currency information.
See ApiCurrencyResource.

Response:

If successful, returns the Currency identifier in the response body and the URL in the response header Location property, or an error message. See ApiCurrencyResource.

Response status codes:

201 The operation was successful and the Currency was created.

Resource URL:

POST /corePatterns/currencies

POST Currency

Inserts a new Currency Rate in an existing Currency.

The following elements are required:

currencyKey (string) The currency logical key.
value (ApiCurrencyCurrencyRatesResource) The ApiCurrencyCurrencyRatesResource instance.
See ApiCurrencyCurrencyRatesResource.

Response:

If successful, this method returns an empty response body.

Response status codes:

204 The operation was successful.

Resource URL:

POST /corePatterns/currencies/{currencyKey}/currencyRates

Resources


CurrencyResource

The following elements are required:

currencyKey (string) The currency logical key.
description (string) The currency description.
fractionDigits (int) Number of digits used to represent fraction values.
currencyUnit (string) The currency main unit (e.g. Euro).
isoCode (string) ISO standard currency code
symbol (string) Symbol used to represent values in the currency (e.g. $)
isExternallyManaged (bool) Is Externally Managed

The following elements are optional:

currencySubUnit (string) The currency sub unit (e.g. Cent).
validFrom (DateTime) Specifies the date from which the currency becomes effective.
validTo (DateTime) Specifies the date from which the currency becomes no longer effective.
useFixedRate (bool) Indicates if this currency uses a fixed rate.
conversionFactor (decimal) Specifies the conversion factor when the currency uses a fixed rate.
pricesFractionDigits (int) Explicit number of digits used to represent fraction values in Prices
externalId (string) External Id
externalVersion (string) External Version
fixedRateReferenceCurrency (string) The currency fixed rate reference currency.
currencyRates (List of CurrencyRateResource) Currency Rates
See CurrencyRateResource.


CurrencyRateResource

The following elements are required:

targetCurrency (string) The target currency.

The following elements are optional:

rate (decimal) The rate value.
